# 碳循环饮食计划生成器

一个基于 Python + PyQt6 开发的碳循环饮食计划生成软件。

## 功能特点

- ✅ 根据身高、体重、年龄、性别计算基础代谢率（BMR）
- ✅ 考虑训练年限和活动水平计算每日总能量消耗（TDEE）
- ✅ 支持减脂、维持、增肌三种目标
- ✅ 自动生成周碳循环计划（高碳/中碳/低碳交替）
- ✅ 提供不同类型日的食物推荐
- ✅ 美观的图形化界面

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python carb_cycling.py
```

## 使用说明

1. **信息输入**：填写个人信息和训练信息
2. **选择目标**：选择减脂、维持或增肌
3. **生成计划**：点击按钮生成个性化饮食计划
4. **查看计划**：切换到"饮食计划"标签查看详细方案
5. **食物推荐**：查看不同类型日的推荐食物

## 碳循环饮食原理

碳循环饮食通过在不同天数交替摄入不同量的碳水化合物：
- **高碳日**：训练日，补充肌糖原，提升训练表现
- **中碳日**：维持日，平衡能量
- **低碳日**：休息日，促进脂肪氧化，提高胰岛素敏感性

## 技术栈

- Python 3.8+
- PyQt6 - 图形界面
- Mifflin-St Jeor 公式 - BMR计算
